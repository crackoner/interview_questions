TAM Pixel Test

Let's test the different ways to integrate an ad conversion pixel into an existing html page. We'll use this page as the page you put the pixel in.

Getting started

  1. Download this page onto your computer and save it as pixel_optimized.html
  2. Save another copy of this page as pixel_timer.html
  3. Download the Outbrain pixel code here.

This is a two-part test.

Part 1: Improve the pixel code you download. Is there any advice you would give to make the code more portable? Rewrite the code with your improvements and embed it in pixel_optimized.html. Test to make sure it runs.

Part 2: Modify the code to fire on the page after the user has been on the page for 15 seconds. Save the code into pixel_timer.html. Test to make sure it runs.

No external javaScript libraries allowed, but you can of course search the net or reach out with questions.

Tip: You can install the code as-is on a test page and see how it fires using the network control panel. Once you know what to look for you can optimize the code and monitor your timer.
